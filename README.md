# Markdown Editor (Offline / Single Page App)

単一のHTMLファイル（と同梱のリソース）で動作する、高機能なMarkdownエディタです。

## 特徴
- **完全オフライン動作**: 外部CDNに依存せず、ローカルのリソースのみで動作します。
- **GFM準拠**: GitHub Flavored Markdown（テーブル、タスクリスト、取り消し線など）に対応。
- **リアルタイムプレビュー**: 左右分割画面で編集結果を即座に確認できます。
- **エディタ/プレビュー比率調整**: マウスドラッグまたは設定画面で、エディタとプレビューの横幅比率を自由に調整できます（20%～80%）。
- **アウトライン表示**: 見出し構造を一覧表示し、素早くナビゲーション。折りたたみ状態も反映され、現在表示中の見出しをハイライト表示。
- **プレビューインライン編集**: プレビュー上で段落、見出し、リスト項目などを直接編集可能。ホバー時に編集ボタンを表示し、直感的な編集体験を提供。
- **テーブル編集機能**: テーブルをプレビュー上で直接編集可能。行追加・削除、セル編集、CSV/TSV貼り付け、セル内改行をサポート。
- **SVG編集機能**: SVGコードブロックをプレビュー上でインタラクティブ編集可能。図形の移動、リサイズ、回転、グループ化、色変更、カスタムツール登録をサポート。
- **検索・置換機能**: 強力な検索機能（正規表現対応、大文字小文字区別、検索対象フィルタリング）と置換機能（単一/全置換、Undo/Redo対応）。
- **見出し折り畳み**: H1〜H6の見出しをクリックしてコンテンツを折り畳めます。
- **Undo/Redo機能**: Ctrl+Z（元に戻す）、Ctrl+Y（やり直す）に対応。最大200回の履歴（設定で変更可能）。
- **キーボードショートカット**: Ctrl+B（太字）、Ctrl+I（斜体）、Ctrl+S（保存）、Ctrl+P（印刷）、Ctrl+F（検索）、Ctrl+H（置換）に対応。
- **PDF出力**: 高品質な画像ベースPDF出力をサポート（ページ番号付き）。
- **印刷機能**: ブラウザの印刷ダイアログでテキスト選択可能なPDFを作成（Ctrl+P）。
- **ドラッグ&ドロップ**: `.md` ファイルをドラッグして読み込めます。
- **Markdownファイル間リンク**: 相対パスで記述された`.md`ファイルへのリンクをクリックすると、新しいタブまたはウィンドウで開きます（設定で選択可能）。
- **コードブロック機能**: Prism.jsによる本格的なシンタックスハイライト、行番号表示、言語名ラベル、コピーボタン搭載（25言語以上対応）。
- **Mermaid図表対応**: フローチャート、シーケンス図、ガントチャート、ER図など、Mermaid記法で図表を描画できます。
- **カスタマイズ**: 本文フォントサイズ、プレビュー横幅、エディタ/プレビュー比率、表ヘッダーやコードエリアの色、コード行間、PDF余白などを設定可能。

### スタンドアロン版（単一HTMLファイル）

`MD_Editor.html` は、すべてのリソースを1つのHTMLファイルにまとめたスタンドアロン版です。

```
.
└── MD_Editor.html       (約3.8MB - これだけで完結)
```

**特徴:**
- ✅ 外部ファイル不要（CSS、JavaScript、ライブラリすべてインライン化）
- ✅ CDN接続不要（完全オフライン動作）
- ✅ 単一ファイルで配布可能
- ✅ MarkdownEditor.html と完全に同じ機能

**ビルド方法:**

```powershell
# PowerShell スクリプトでビルド
.\build.ps1
```

詳細は [docs/build_process.md](docs/build_process.md) を参照してください。

## 使い方

1. **起動**: `MarkdownEditor.html` をブラウザ（Chrome, Edge, Firefox, Safari）で開きます。
2. **編集**: 左側のペインに入力すると、右側にプレビューされます。
3. **保存**: 
   - **Ctrl + S** (Macは Cmd + S) で上書き保存（対応ブラウザのみ）またはダウンロード保存ができます。
   - ツールバーの「保存」ボタンでも可能です。
4. **キーボードショートカット**:
   - **Ctrl + Z** (Macは Cmd + Z): 元に戻す（Undo）
   - **Ctrl + Y** (Macは Cmd + Y): やり直す（Redo）
   - **Ctrl + B** (Macは Cmd + B): 選択テキストを太字化/解除
   - **Ctrl + I** (Macは Cmd + I): 選択テキストを斜体化/解除
   - **Ctrl + F** (Macは Cmd + F): 検索ダイアログを開く
   - **Ctrl + H** (Macは Cmd + H): 置換ダイアログを開く
   - **F3**: 次を検索 / **Shift + F3**: 前を検索
   - **Esc**: 検索ダイアログを閉じる
5. **PDF出力**: ツールバーの「PDF保存」ボタンで画像ベースPDFを出力、「印刷」ボタン（Ctrl+P）でテキスト選択可能なPDFを作成できます。
6. **ファイル読み込み**: `.md` または `.txt` ファイルをウィンドウにドラッグ&ドロップしてください。
7. **Mermaid図表**: コードブロックで言語を `mermaid` に指定すると、図表として描画されます。
   ```markdown
   ```mermaid
   graph TD
       A[開始] --> B{条件}
       B -->|Yes| C[処理1]
       B -->|No| D[処理2]
   ```
   ```

## 技術仕様
- **Markdown Parser**: marked.js v11.0.0
- **Sanitizer**: DOMPurify v3.0.6
- **PDF Generator**: jsPDF v2.5.1 + html2canvas v1.4.1
- **Diagram Rendering**: Mermaid.js v10.6.1
- **Syntax Highlighting**: Prism.js v1.29.0（25言語以上対応）
- **SVG Manipulation**: SVG.js v3.2.x（プラグイン: draggable, select, resize）
- **Color Picker**: Pickr v1.9.0（透明度対応）
- **Storage**: localStorage (設定保存)、IndexedDB (ディレクトリハンドル保存)
